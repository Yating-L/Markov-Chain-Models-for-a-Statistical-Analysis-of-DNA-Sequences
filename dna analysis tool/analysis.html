About the tool

This tool is used to identify overrepresented and underrepresented motifs with user defined length. Motifs are sorted by their z-scores calculated by Markov Chains Model, M1 to M4.

To do the DNA analysis, you have to set Markov Chain Order, maximum word length, minimum word length and input file(s). For detailed instruction and examples, see _____. The results will output to the screen after you click submit. Be patient if your file is big.
